Welcome to Stackstats documentation!¶

This is small documention for StackStats. This program is part of Python-Assigment. It use the API of Stackoverflow to get some answer and comment data and calculate some statistics.

> Retrieve the StackOverflow answer data for a given date/time range from the StackExchange API (https://api.stackexchange.com/docs/answers).

> Retrieve the comment data for a given set of answers (https://api.stackexchange.com/docs/comments-­on-answers).

> For a given date/time range calculate:

  • the total number of accepted answers.
  • the average score for all the accepted answers.
  • the average answer count per question.
  • the comment count for each of the 10 answers with the highest score.
  • Collect and return the calculated statistics in tabular, HTML or JSON form

Installation and Run Instruction¶

1
2
cd stackstats/
sudo python setup.py install

Run examples¶

1
2
stacks (-h | --help)
stacks --stats --since '2016-06-02 10:00:00' --until '2016-06-02 11:00:00' --output-format json

Warning

This program tries to get all the pages data that API send to us. If the package of data has ‘has_more’ at the end of the package it send a new request. This is the main reason that this programm is delayed to print the results.

1
2
3
4
if 'has_more' in response and response['has_more']:
    params["page"] += 1
else:
    break
